# kakitori_quest Phase 1 設計仕様書

作成日: 2026-05-27

---

## 1. プロジェクト概要

`@k1low/kakitori` を使った書き取り練習ゲーム。文字を正確に書くことでキャラクターを召喚し、崩れた文字モンスターと戦うドラクエ風RPG。

**リポジトリ名:** `kakitori_quest`  
**対象プラットフォーム:** タブレットブラウザ（静的Webアプリ）  
**Phase 1 対象文字:** ひらがなのみ

---

## 2. ビジュアルスタイル

- **テイスト:** ドラクエ風レトロRPG
- **配色:** 黒背景 ＋ 白枠ウィンドウ
- **フォント:** ドット絵日本語フォント（マルモニカ / PixelMplus / 美咲フォント 等、商用利用可のもの）
- **キャラクター表現:** 文字そのものがキャラクター・敵として機能する（専用スプライト不要）
- **敵の見た目:** 同じ文字を反転・グリッチ加工して「崩れた文字」として表現

---

## 3. ゲームループ（1ステージ = 1単語）

```
① ステージ選択
   └ 単語リストからひらがな単語を選ぶ（例: 「いぬ」）

② 1文字目を書く（書き取りエリア）
   ├ 文字ガイドが薄く表示される
   └ 筆跡がオーバーレイ表示される

③ 筆順ミス発生時
   └ ハート ×1 消費 → 同じ画を再挑戦

④ 文字完成
   └ とめ/はね/はらいの精度 → 召喚キャラの強さが確定
      ├ 高精度 → 強い文字キャラ
      └ 低精度 → 弱い文字キャラ

⑤ バトル（オート決着）
   ├ 召喚した文字キャラ vs 崩れた同文字モンスター
   ├ 勝利 → 次の文字へ進む
   └ 敗北 → ハート ×1 消費 → 同じ文字を再挑戦

⑥ ハート0になった場合
   └ 単語の最初からリスタート（ハートは3に回復）

⑦ 単語内の全文字クリア
   └ ステージクリア → スター評価（★1〜3）表示
```

---

## 4. バトル仕様

### 4.1 勝敗判定

| 結果 | 条件 | 処理 |
|---|---|---|
| 勝利 | とめ/はね/はらい精度が閾値以上 | 次の文字へ進む |
| 敗北 | とめ/はね/はらい精度が閾値未満 | ハート消費、同じ文字を再挑戦 |

- 勝敗は二択（グラデーションなし）
- 精度閾値の具体値は kakitori API の精度データを確認後に決定

### 4.2 ハート（HP）仕様

| 項目 | 仕様 |
|---|---|
| 初期ハート数 | 3 |
| ハート消費タイミング | 筆順ミス時 / バトル敗北時 |
| ハート0時の処理 | 単語の最初からリスタート（ハート3に回復） |
| 将来拡張 | レベルとステージ難易度に応じてハート数を可変にする（Phase 2以降） |

### 4.3 2軸の役割分担

| 軸 | 判定元 | 影響先 |
|---|---|---|
| 「正しく書けるか」 | 筆順の正解/不正解 (`onMistake`) | ハート消費 |
| 「丁寧に書けるか」 | とめ/はね/はらいの精度 | 召喚キャラの強さ → バトル勝敗 |

## 5. 画面構成

### 5.1 インゲーム画面（分割画面）

```
┌─────────────────────┬──────────────────┐
│   バトルステージ     │   書き取りエリア  │
│                     │                  │
│  [崩れた文字]       │  ┌────────────┐  │
│       vs            │  │ ガイド薄表示│  │
│  [召喚キャラ]       │  │ 筆跡オーバー│  │
│                     │  └────────────┘  │
│  メッセージウィンドウ│                  │
│  コマンド窓         │  ❤️ ❤️ ❤️        │
└─────────────────────┴──────────────────┘
```

- 書き取りエリアの位置は**設定で左・右・下に切り替え可能**（利き手・タブレット向き対応）

### 5.2 画面遷移

```
タイトル画面
  └→ ステージ選択画面
       └→ インゲーム画面（書き取り + バトル）
            ├→ ステージクリア画面（★評価表示）
            │    └→ ステージ選択画面
            └→ ゲームオーバー画面（ハート0）
                 └→ 同ステージからリスタート
```

---

## 6. スター評価

| 評価 | 条件 |
|---|---|
| ★★★ | とめ/はね/はらい精度が高い（全文字で高品質） |
| ★★ | とめ/はね/はらい精度が中程度 |
| ★ | クリアしたが精度が低い |

- スター評価の具体的な閾値は kakitori API のデータ確認後に決定

---

## 7. 単語リスト（Phase 1）

- **方針:** 手作りの固定リスト 30〜50語
- **対象:** ひらがな2〜4文字の日常語
- **例:** いぬ / ねこ / うみ / やま / かわ / はな / そら / ほし / つき / てり / など
- **将来拡張:** Phase 2 以降で JMdict（CC BY-SA-4.0）を使った学年別自動生成に移行

## 9. Phase 1 スコープ境界

### 含むもの

- ひらがな書き取り
- 手作り単語リスト 30〜50語
- 3ハート制（筆順ミスで消費）
- とめ/はね/はらい精度でバトル勝敗・★評価
- 崩れた文字キャラ vs 召喚キャラ（オート決着）
